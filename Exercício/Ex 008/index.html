Itálico ou ênfase

Aqui é a mesma coisa, eu tenho duas opções. Ou faço utilizando a TAG <I> (não semântica), ou posso fazer utilizando a TAG <EM> (semântica).

Texto grande e pequeno

Pra aumentar o texto o correto é fazer no CSS, mas posso fazer no HTML utilizando a TAG <BIG>. E para deixar menor, posso utilizar a TAG <SMALL>.
